# KMDev — AI Sales & Client Intake Agent (concept note)

> Status: **IDEA / NOT YET IMPLEMENTED.** Recorded 2026-06-12. Do not build until requested.

## Vision
Upgrade the portfolio chatbot from a Q&A bot into a **virtual project consultant** — an "AI Sales & Client Intake Agent." Instead of just answering questions, it guides a visitor from *"Order Project"* through quote estimation and requirements gathering to a **developer-approved** proposal, feeding directly into the existing Client Management / agency admin system. Goal: make the portfolio feel like a real software agency platform, not a freelancer site.

## Entry point
User clicks **"Order Project"**. AI greets:
> Hi 👋, I'm the project assistant. I can help you request a quote, estimate pricing, gather requirements, and prepare a project proposal for review.

Then offers options:
1. Learn About Me
2. View Services
3. Request a Quote
4. Start a New Project
5. Ask a Question
6. Existing Client Support

## Flow: "Start a New Project"

**Step 1 — Contact Details**
Ask: Full Name, Email Address, Phone Number, Company Name (optional).
Notice: *"Your contact information is required for project communication, quotations, approvals, and legal documentation. Your information will be handled according to our privacy policy."*

**Step 2 — Project Discovery**
"What would you like to build?" → Website, Mobile App, Marketplace, AI Assistant, Admin Dashboard, Custom Software.

**Step 3 — Smart follow-up questions** (LLM-driven, e.g. Gemini / Grok)
e.g. for *"I need a marketplace app"*:
- Will users create accounts?
- Will there be payments?
- Will products be public or private?
- Do you need an admin dashboard?
- Do you need delivery support?
- Do you need AI features?

**Step 4 — AI Summary**
Bullet list of requirements + **Estimated Complexity** (Low / Medium / High) + **Estimated Duration** (e.g. 4–8 weeks).

**Step 5 — Preliminary Quote** (itemized)
Example:
- Base Website Development — R2,000
- Admin Dashboard — R800
- Marketplace Features — R1,500
- AI Assistant — R350 setup
- Monthly Maintenance — R250/month
- **Estimated Total — R4,650**

Must clearly display: *"⚠️ This is an AI-generated estimate. Final pricing requires developer approval."*

**Step 6 — Admin Approval Required (CRITICAL)**
The AI must **never** automatically finalize pricing.
- Status: **Pending Developer Review**
- Admin receives a notification + dashboard card: Client, Estimated total, **[Approve] [Modify] [Reject]**.

**Step 7 — Auto-generated documents (on approval)**
Generate: Quote, Requirements Specification, Proposal, Timeline, Contract Draft.
All marked **"Generated by AI — Pending Developer Approval"** until the developer approves.

**Step 8 — WhatsApp & Email notifications (after approval)**
Client receives:
> Your quote has been approved. Reference: KMDEV-2026-00045

with links: **View Quote**, **View Contract**, **Book Consultation** — via Email and WhatsApp.

## AI knowledge sources (grounding / training)
Portfolio projects, Services, Pricing rules, FAQs, Client workflow, Contracts, Maintenance plans.
Enables questions like:
- "How much is an e-commerce website?"
- "Do you offer maintenance?"
- "How long does development take?"
- "What projects have you built?"

## Favorite addition — 🎙 Talk to AI
A voice button. Users speak instead of typing; the AI collects requirements by voice, generates the summary, and submits everything to the admin dashboard.

## Integration notes (for whoever builds it later)
- Reuses existing models: `Client`, `ClientProject`, `ClientDocument` (reference format `KMDEV-YYYY-XXXXX`), pricing options/extras in `AgencySettings`.
- Intake should create a `Client` + `ClientProject` in a Lead/Consultation stage with status **Pending Developer Review**; documents generated as drafts with a "Generated by AI" flag until the admin approves.
- Touch points already in the codebase: chat API `src/app/api/chat/route.ts`, chatbot UI `src/components/chatbot/Chatbot.tsx`, knowledge `src/lib/chatbot-knowledge.ts`.
- Not yet built and would be new work: LLM follow-up logic, AI estimate engine, "Generated by AI / Pending Approval" document state, admin approve/modify/reject on AI quotes, WhatsApp + Email notifications, voice intake.
